# Flashcards
The app for learning words in multiple languages
## Info:
- Java 11
- Gradle 7.5
- Jackson
## About App
The application was created to make learning foreign languages easier. The application currently has 5 languages. These are Polish, English, German, Spanish and Norwegian. 
<br/>
 <br/>

 <br/>
 <br/>
In the language selection menu it is possible to select languages such as
- Polish -> English 
- English -> Polish 
- Polish -> German 
- German -> Polish 
- Polish -> Spanish 
- Spanish -> Polish 
- Polish -> Norwegian 
- Norwegian -> Polish
<br/>
  <img src="https://github.com/ReFZero/Flashcards/assets/82156185/ee062214-34ac-4411-8a8c-ab051c59fa08"  width="25%" height="25%"/>
<br/>
<br/>
In the menu you can choose between many sets of words. This number may vary depending on the language.
<br/>
<br/>
  <img src="https://github.com/ReFZero/Flashcards/assets/82156185/1ef92272-3a88-4701-85c5-fbc2d1e80320"  width="25%" height="25%"/>
<br/>
<br/>
After selecting a specific set of words, we will have the opportunity to learn 50 words. The application allows you to draw words from a set and change the language of the flashcard by pressing the appropriate button. There is a word number in the upper left corner. The word is displayed in the center. At the bottom there are buttons to control the flashcards.
